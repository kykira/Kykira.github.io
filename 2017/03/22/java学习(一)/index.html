一、基本语法和标识符

在java语言的基本语法中，有以下几点是需要注意的

1.Case Sensitivity - 案例敏感性，也就是说，java语言中是区分大小写的，这意味着标识符Hello和hello具有着不同的含义，这与大多数编译语言一样。

2.Class Names - 类名，定义所有类名，它的首字母都应该大写，而且也应该是由有实际含义的单词组成，而且内部的每个单词的首字母也应该大写。

3.Method Names - 方法名称，与类名差不多，命名由有实际含义的单词组成，不过名称要以小写字母开头，每个内部单词首字母大写。

4.Program File Name - 程序的文件名称应该和类名完全一致，假设’MyFirstJavaProgram’是类名。那么文件应该保存为’MyFirstJavaProgram.java’。

5.public static void main（String args []） - Java处理程序从main（）方法开始，除了接口类和枚举类，每个程序都应该有该方法用于调试或者执行。

Java中用于类，变量和方法的名称称为标识符，标识符也有几点要注意一下的地方，当代码熟练以后便不会犯这种小错误了。

1.所有标识符应以字母（A到Z或a到z），货币字符（$）或下划线（_）开头。

2.标识符区分大小写。

3.关键字不能用作标识符。

二、数据类型和运算符

Java支持8种基本数据类型，基本数据类型不支持面向对象的编程机制，也就不具备“对象”的特性：没有属性和方法可以调用。由于Java是面向对象的编程语言，所以Java提供了基本数据类型的包装类，包装类自然提供了各种方便的方法，而在进行简单、有效的常规数据处理中，用这8种基本数据类型足够了。包装类的使用会在下一章的类和对象中有所提及（大概）。
