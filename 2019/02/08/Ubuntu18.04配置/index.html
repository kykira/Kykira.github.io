Ubuntu 18.04 LTS 安装搭建配置

发表于 2019-02-08 | 分类于 折腾

既然有一台闲置surface pro4，那就装个Ubuntu系统玩吧

基本系统配置

第一次更新

sudo apt-get update

所有软件→设置→区域和语言→管理已安装的语言，完成语言列表的更新。

surface设备内核更换

https://github.com/jakeday/linux-surface

ubuntu下面 将汉字桌面、下载 换成 英文

export LANG=en_US

执行

xdg-user-dirs-gtk-update

此时，在位置里看到的常用中文目录已经换成英文目录。只需将原中文目录的内容拷贝到相应英文目录，并删除中文目录即可。

然后再执行

export LANG=zh_CN.UTF-8

更换终端oh-my-zsh:

sudo apt-get install git

sudo apt-get install zsh

wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh

chsh -s /usr/bin/zsh

更换终端主题

安装 fonts-powerline字体

sudo apt-get install fonts-powerline

https://github.com/agnoster/agnoster-zsh-theme

vim ~/.zshrc 修改 ZHS_THEME=”agnoster”

修改agnoster.zsh-theme文件，把AGNOSTER_PROMPT_SEGMENTS中prompt_context删除去掉用户和机器信息

修改终端字体为:”meslo lg s dz for powerline regular”

测试AGNOSTER_PROMPT_SEGMENTS[2]=

系统美化

  • Gnome-tweak-tool：sudo apt install gnome-tweak-tool
  • 自定义 Shell ：所有软件(Win+A)->Ubuntu 软件->搜索 “User Themes”->安装扩展

  • Gnome 扩展

    Ubuntu 18.04 抛弃了 Unity 桌面转而使用 Gnome ，所以 Gnome 桌面下的一些 Shell 扩展在 Ubuntu 18.04 中就可以使用了。

    在Gnome-tweak-tool一节中就提到通过 Ubuntu 软件中心下载、安装和管理 Gnome 扩展（附加组件），下面介绍一种通过浏览器对 Gnome 插件的安装管理方式：

    首先安装 Chrome Gnome Shell ：sudo apt install chrome-gnome-shell

    • user Themes
    • Alternate Tab
    • hide top bar
    • dash to dock
    • internet speed meter
    • random walls
    • focus my window

Ubuntu桌面无法显示托盘图标：

安装TopIconPlus的gnome-shell扩展，

命令：sudo apt-get install gnome-shell-extension-top-icons-plus gnome-tweaks，然后用r命令重启gnome-shell，最后用gnome-tweaks开启这个扩展。

主题：Ultimate-Maia-Green

字体：思源黑体light

常用软件

中文输入法-搜狗

  • 安装 Fcitx 框架：sudo apt install fcitx
  • 下载输入法：搜狗 for Linux
  • 激活框架：设置->区域和语言->管理已安装的语言->键盘输入法系统：fcitx
  • 选择搜狗输入法：所有软件（Win+A）-> Fcitx 配置->搜狗输入法 （多半找不到那就重启）

QQ/WeChat

  • 下载环境：git clone https://github.com/wszqkzqk/deepin-wine-ubuntu.git

  • 执行安装：./install.sh

  • 安装deepin.com应用容器：在http://mirrors.aliyun.com/deepin/pool/non-free/d/中下载想要的容器，点击deb安装即可。以下为推荐容器:

    • QQ：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.im/

  • 1，先退出deepin-tim或deepin-qq，否则会提示错误。

    2，运行env WINEPREFIX="$HOME/.deepinwine/Deepin-TIM" winecfg
